# CCAT Practice Quiz

A comprehensive practice test for the Criteria Cognitive Aptitude Test (CCAT) used by Crossover and other companies.

## Features

- **50 Questions** covering Math, Logic, and Verbal Reasoning
- **15-minute timer** matching real CCAT conditions
- **Keyboard shortcuts** - Press A, B, C, or D to answer quickly
- **Auto-advance** to next question after selection
- **Score tracking** in real-time
- **Review mode** to see correct answers after completion
- **Breakdown by category** to identify strengths and weaknesses

## How to Use

1. Open `index.html` in your web browser
2. Click "Start Test" to begin
3. Answer questions by:
   - Clicking on options, OR
   - Pressing A, B, C, or D keys for quick selection
4. Navigate using:
   - Next/Previous buttons
   - Arrow keys (← →)
5. Complete all 50 questions or wait for timer to expire
6. Review your results and see which questions you got right/wrong

## Question Types

- **Math (20 questions)**: Algebra, percentages, sequences, geometry
- **Logic (15 questions)**: Pattern recognition, deductive reasoning, analogies
- **Verbal (15 questions)**: Vocabulary, synonyms, antonyms, word relationships

## Tips for CCAT Success

1. **Speed matters** - You have 18 seconds per question on average
2. **Don't get stuck** - Skip difficult questions and come back if time permits
3. **Use keyboard shortcuts** - Much faster than clicking
4. **Practice regularly** - Familiarity with question types helps
5. **Read carefully** - Many questions have tricky wording
